# ll2dot

[![GoDoc](https://godoc.org/github.com/mewrev/ll2dot?status.svg)](https://godoc.org/github.com/mewrev/ll2dot)

ll2dot is a tool which converts LLVM IR assembly files to GraphViz DOT files.
